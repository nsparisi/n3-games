-Download Codeblocks w/ MinGW
-In the codeblocks install location, find the MinGW version number
-Use this forum to get the correct SFML build. 
	http://en.sfml-dev.org/forums/index.php?topic=9513.0
	correct = MinGW TDM GCC Version that matches Codeblocks MinGW version
-Add the SFML build folder in with the rest of the CB project
-now follow http://www.sfml-dev.org/tutorials/2.0/start-cb.php
	-should have to include /lib, /include, and link libraries
-finally add dlls from sfml/bin into the same location that the .exe is built to. (i.e. bin/debug/)
-to get opengl to work add the following under "build options->Linker settings->Other linker options"
	-lsfml-graphics-d
	-lsfml-window-d
	-lsfml-audio-d
	-lsfml-system-d
	-lopengl32
	-lglu32
